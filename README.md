# 🩺 AI Diabetes Risk Predictor

A machine learning-based system to predict diabetes risk with an interactive dashboard and AI assistant.

## 🚀 Features
- Diabetes prediction using Random Forest
- Risk level classification (Low, Medium, High)
- Doctor recommendation + precautions
- Data visualization (graphs, heatmap)
- Model comparison
- Patient history storage
- AI Assistant with voice output

## 🛠️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas & NumPy
- Plotly

## ▶️ Run Project

```bash
streamlit run diabetes_app.py
